C++判定某年是否为闰年

2019-02-01 C++

开发工具：VS 2017

// 闰年判定条件： 1、能被4整除，但不能被100整除； 2、能被400整除。

#include "pch.h"
#include <iostream>
using namespace std;

void main()
{
	int year;
	cout << "请输入一个年份：" << endl;
	cin >> year;
	if ((year % 4 == 0 && year % 100 != 0) || year % 400 == 0)
	{
		cout << "这是闰年。" << endl;
	}
	else {
		cout << "这不是闰年。" << endl;
	}
}
